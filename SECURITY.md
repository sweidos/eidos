# Security Policy

## Supported Versions

Only the latest published version of `@sweidos/eidos` on npm receives security fixes.

| Version  | Supported |
| -------- | --------- |
| latest   | ✅        |
| < latest | ❌        |

## Reporting a Vulnerability

Please **do not open a public issue** for security vulnerabilities.

Instead, report it privately via [GitHub Security Advisories](https://github.com/iamadi11/eidos/security/advisories/new) for this repository.

Include as much detail as possible:

- A description of the vulnerability and its impact
- Steps to reproduce or a proof-of-concept
- The affected version(s)

We aim to acknowledge reports within 72 hours and release a fix or mitigation as soon as possible.
